## 📖 项目简介

本项目提供基于 **GitHub Actions** 的自动化内核编译工作流，支持 **OnePlus** 和 **Realme** 多款搭载 **SM8750** 平台的设备。通过高度集成的脚本，实现一键编译包含 **(Re)SukiSU**、**SUSFS**、**风驰调度** 等功能的 OKI 内核。
